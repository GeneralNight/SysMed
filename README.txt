Para que o sistema execute corretamnete, copie todo o conteúdo da pasta para a pasta do servidor;

Caso esteja usando um servidor local, como WAMP ou XAMPP, coloque na pasta que corresponde a "pasta servidor";

No WAMP é a pasta www;

No XAMPP é a pasta htdocs;


Não se esqueça de fazer a importação do banco de dados e configurar o arquivo "Config.php"* com as configurações do seu servidor;


Depois basta acessar a pasta "sysmed" a partir do servidor.


* O arquivo se localiza em "PHP/Config.php".
